# Cybersecurity Anomaly Detection with Transfer Learning

LSTM-based cybersecurity anomaly detection using transfer learning.

## Project Overview
This project uses a pre-trained LSTM model to detect anomalies in cybersecurity data using transfer learning. By freezing earlier layers and retraining the final layer, the model improves performance while reducing training time.
